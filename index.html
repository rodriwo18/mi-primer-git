Comandos para git:

  1. Primer comando: git init (para inicializar git, 1 sola vez por proyecto)
  2. Segundo comando: git status (Para checkear el estado en git en nuestro proyecto
  3. Tercer Comando: git add (Para trackear todos los archivos)
  4. Cuarto comando: git commit -m "Mensajito" (Para hacer el paquetito de una version, que podemos subir a la nube)
